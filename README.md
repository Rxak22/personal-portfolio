Full portfolio project with advanced features:
- Smooth animations (AOS)
- GSAP intro animation
- three.js hero 3D TorusKnot + particles
- Animate-on-hover project cards
- Sticky animated navbar + mobile drawer
- Blog section
- Dark / Light theme toggle
- Contact form wired to Formspree

To run:
1. npm install
2. npm run dev

Notes:
- three.js is loaded via unpkg in the Hero component (ES module). For offline dev, consider installing 'three' as dependency.
- AOS is loaded via CDN. GSAP used via CDN.
